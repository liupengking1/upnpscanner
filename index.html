Upnp Scanner

UPnP Scanner can discover any UPnP devices present in the local network. Features

  • Discover all UPnP devices in the network
  • Discover DIAL devices including Chromecast and Fire TV
  • Filter device by device type
  • Automatic update when devices are added/removed from the network
  • Device details for selected devices ** The application is based on the Cling - Java/Android UPnP library http://4thline.org/projects/cling/
